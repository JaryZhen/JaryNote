### Netty
 ###### 2017.11.30 - 2017.12.18
      
      1）本质：JBoss做的一个Jar包
      2）目的：快速开发高性能、高可靠性的网络服务器和客户端程序
      3）优点：提供异步的、事件驱动的网络应用程序框架和工具
      
      通俗的说：一个好使的处理Socket的东
      如果没有Netty？
      远古：java.net + java.io
      近代：java.nio
      其他：Mina，Grizzly
      Netty提供异步的、事件驱动的网络应用程序框架和工具，用以快速开发高性能、高可靠性的网络服务器和客户端程序。
